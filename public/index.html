Get All Notes

Endpoint URL: /api/notes

A GET request to this endpoint returns a list of notes in the database. Each note has the following fields – id, title, textBody, tags

Add a New Note

Endpoint URL: /api/notes Method: POST

Make a post request to the endpoint in order to add a new note. The post request must have content-type set to application/json and the request body must contain a title and a textBody. Optionally you can also specify tags associated with the notes. In order to add tags, include an array of strings in the request body with the key tags. If the operation was successful, it will return a message and the id of the new note.

Example Usage 
POST http://localhost:4000/api/notes HTTP/1.1
Content-Type: application/json

{
  "title": "Note 5",
  "textBody": "The quick brown fox jumps over the lazy dog",
  "tags": ["Entertainment", "Pets"]
}
Example Output 
{
  "message": "Note added successfully",
  "id": 8
}

Get Note by ID

Endpoint URL: /api/notes/:id

Use this endpoint to get details for one note specified by ID. A get request to /api/notes/:id where :id refers the unique id of the note will yield the following results.

Example Output 
{
  "id": 5,
  "title": "Note Five",
  "textBody": "The quick brown fox jumps over the lazy dog",
  "tags": [
    "Entertainment",
    "Pets",
    "Dogs"
  ]
}

Update Note by ID

Endpoint URL: /api/notes/:id Method: PUT

Use this endpoint to update details of a note specified by ID. Make a PUT request to /api/notes/:id, where :id refers to the ID of the note you'd like to update. At least one of the fields must be specified in order to successfully update the note.

Example Usage 
PUT http://localhost:4000/api/notes/6 HTTP/1.1
Content-Type: application/json

{
  "tags": [
    "Entertainment",
    "Pets",
    "Dogs"
  ]
}
Example Output 
{
  "message": "Note with id 6 updated successfully"
}

Delete Note by ID

Endpoint URL: /api/notes/:id Method: DELETE

Use this endpoint to delete a note specified by ID. This action is irreversible. Make a DELETE request to /api/notes/:id, where :id refers to the ID of the note you'd like to delete. Upon successful deletion you will receive the following response.

Example Usage 
DELETE http://localhost:4000/api/notes/7 HTTP/1.1
Example Output 
{
  "message": "Note with id 7 has been deleted successfully"
}
